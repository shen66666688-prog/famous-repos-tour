## 3 Interesting Folders

### 1. drivers

This folder contains hardware drivers for many devices such as keyboards, GPUs, network cards, and storage systems. It demonstrates how Linux supports a huge variety of hardware.

### 2. kernel

This directory contains the core kernel logic, including process scheduling and system management. It is the heart of the Linux operating system.

### 3. net

This folder handles networking features such as TCP/IP communication, sockets, and network protocols. It shows how Linux powers internet communication.
